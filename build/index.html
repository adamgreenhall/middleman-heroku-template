Generate a static site with middleman and then host it with heroku. Use clean urls.

To make your site use:

  • haml and textile
  • sass and compass
  • coffeescript and sprockets
  • livereload (middleman server --livereload)

Then when you are ready, put the site on heroku:

  • heroku create (the first time)
  • middleman build
  • git add build/* (if you add new pages)
  • git commit
  • git push heroku master
